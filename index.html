1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

  • JavaScript (JS) je interpretační programovací jazyk, který se používá pro vytváření interaktivních webových aplikací.
  • Umožňuje upravovat obsah webových stránek v reálném čase (například aktualizace bez obnovení stránky).
  • Umožňuje reagovat na události jako jsou kliknutí, pohyby myši, klávesové vstupy atd.
  • JavaScript umožňuje dynamicky měnit obsah a strukturu HTML dokumentu. Může přidávat nebo odstraňovat prvky stránky, měnit jejich styly nebo reagovat na uživatelské vstupy.

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Use let, const, or (less commonly) var
  2. Stick to consistent indentation (usually 2 or 4 spaces). Proper indentation makes the code more readable.
  3. When writing complex code, it’s good to use comments. Comments are written either as single-line (//) or multi-line (/* */).
  4. Use meaningful names. It's recommended to use camelCase for variables and functions (e.g., myFunction), while class names use PascalCase (e.g., MyClass).
  5. Although JavaScript automatically inserts semicolons, it is recommended to use them explicitly.

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Zobrazí jednoduché varovné okno s textem a tlačítkem „OK“.*/
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
let password = prompt('Zadej heslo:');

confirm()

/* Zobrazí okno s otázkou a dvěma tlačítky: „OK“ a „Zrušit“.*/
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování:
Popup okna často narušují přirozený tok stránky.
Popup okna mohou být snadno zneužita k vytváření falešných dotazů na uživatele, například pro získání osobních údajů.
